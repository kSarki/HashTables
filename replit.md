# Schedule Search Program

## Overview
A Python command-line application for managing and searching schedule items loaded from CSV files.

## Project Structure
- `schedule_item.py` - ScheduleItem class definition
- `schedule.py` - Schedule class for managing collections of schedule items
- `main.py` - Main program with CSV loading and interactive menu
- `schedule_data.csv` - Sample schedule data

## Recent Changes
- 2025-11-17: Initial project setup with Python 3.11

## Features
- Load schedule data from CSV files
- Search and filter schedule items
- Interactive command-line menu
- Display formatted schedule results
